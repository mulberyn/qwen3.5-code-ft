```python
def group_by_length(strings):
    """
    将字符串列表按字符串长度分组，返回一个字典。
    
    参数:
    strings (list of str): 输入的字符串列表
    
    返回:
    dict: 键为字符串长度，值为对应长度的字符串列表的字典
    """
    grouped_strings = {}
    for string in strings:
        length = len(string)
        if length not in grouped_strings:
            grouped_strings[length] = []
        grouped_strings[length].append(string)
    return grouped_strings

# 示例用法
input_strings = ["apple", "bat", "cat", "dog", "elephant", "fish"]
result = group_by_length(input_strings)
print(result)
```

**时间复杂度**: O(n)，其中 n 是输入字符串列表的长度。因为每个字符串只会被遍历一次。

**空间复杂度**: O(n)，因为我们需要存储所有字符串在结果字典中。
